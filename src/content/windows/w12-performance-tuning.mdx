性能优化在 Windows 和 macOS 上基本一致。

## Windows 特有优化

### 文件路径最大长度

```jsonc
{
  // Windows 长路径支持
  "files.watcherExclude": {
    "**/node_modules/**": true,
    "**/.git/objects/**": true
  }
}
```

### 防病毒排除

如果使用 Windows Defender，建议将以下目录加入排除列表：
- `%APPDATA%\Code`
- 项目中的 `node_modules`
