设置功能跨平台通用。以下仅列出 Windows 特有的差异。

## 配置文件路径

| 配置 | Windows 路径 |
|------|-------------|
| 用户设置 | `%APPDATA%\Code\User\settings.json` |
| 快捷键 | `%APPDATA%\Code\User\keybindings.json` |
| 用户片段 | `%APPDATA%\Code\User\snippets\` |

## 快捷键差异

Windows 使用 `Ctrl` 替代 macOS 的 `Cmd`，`Alt` 替代 `Option`。

```jsonc
// keybindings.json（Windows）
[
  {
    "key": "ctrl+e",
    "command": "workbench.action.quickOpen"
  },
  {
    "key": "alt+ctrl+t",
    "command": "workbench.action.terminal.toggleTerminal"
  }
]
```
