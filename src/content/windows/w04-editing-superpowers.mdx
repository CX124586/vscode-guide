编辑功能在 **Windows 和 macOS** 上完全一致，仅快捷键不同（`Ctrl` 替代 `Cmd`）。

## 功能一览

| 功能 | macOS | Windows |
|------|-------|---------|
| 多光标添加 | `Cmd+D` | `Ctrl+D` |
| 全选匹配 | `Cmd+Shift+L` | `Ctrl+Shift+L` |
| 列选择 | `Option+Shift+拖拽` | `Alt+Shift+拖拽` |
| 行移动 | `Option+↑/↓` | `Alt+↑/↓` |
| 格式化文档 | `Shift+Option+F` | `Shift+Alt+F` |
| 行注释 | `Cmd+/` | `Ctrl+/` |
| 块注释 | `Option+Shift+A` | `Alt+Shift+A` |
| 重命名符号 | `F2` | `F2` |
| 触发补全 | `Ctrl+Space` | `Ctrl+Space` |

## 详细教程

所有编辑技巧（多光标、IntelliSense、Snippets、Emmet、格式化、折叠等）均为 **跨平台通用**，操作方式完全一致。

👉 详见 [macOS 编辑超能力](/docs/macos/04-editing-superpowers)
