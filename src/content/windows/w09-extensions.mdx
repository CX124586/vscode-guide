扩展在 Windows 和 macOS 上**完全相同**。

## 扩展管理

| 操作 | Windows |
|------|---------|
| 打开扩展面板 | `Ctrl+Shift+X` |
| 安装扩展 | 点击 Install |
| 命令行安装 | `code --install-extension <id>` |

## 推荐扩展

所有扩展推荐（Prettier、ESLint、GitLens、语言支持、主题）均为 **跨平台通用**。

👉 详见 [macOS 扩展生态](/docs/macos/09-extensions)
