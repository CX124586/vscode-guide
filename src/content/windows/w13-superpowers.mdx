Superpowers AI 工作流在 Windows 和 macOS 上**完全一致**。

## Windows 安装注意事项

### 前置条件

- VS Code 1.90.0+
- GitHub Copilot 扩展（已安装并激活Chat）
- Git for Windows（[下载](https://git-scm.com/download/win)）

### 安装

1. 在 VS Code 中打开 Chat（`Ctrl+Shift+I`）
2. 点击设置 → **Plugins**
3. **"从源安装插件"** → 输入：
   ```
   https://github.com/obra/superpowers
   ```
4. 验证：`@superpowers /help`

## 详细教程

所有 Superpowers 工作流（`/plan`、`/tdd`、`/debug`、`/review`、`/finish`）均为 **跨平台通用**。

👉 详见 [macOS Superpowers AI 工作流](/docs/macos/13-superpowers)
