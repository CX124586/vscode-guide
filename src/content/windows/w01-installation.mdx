## 系统要求

VS Code 支持 **Windows 10（1909+）**、**Windows 11** 及更高版本，支持 x64 和 ARM64 架构。

## 安装方式

### 方式一：官网下载（推荐）

1. 访问 [VS Code 官网](https://code.visualstudio.com)
2. 点击 **Download for Windows**
3. 运行下载的 `VSCodeSetup-x64-*.exe` 安装程序
4. 按照安装向导操作，**强烈建议勾选**以下选项：

<img src="/images/windows-installer.svg" alt="Windows 安装程序选项" class="w-full max-w-lg mx-auto my-4 rounded-lg shadow-lg" />

5. 完成安装后启动 VS Code

### 方式二：winget 安装（Windows 11 推荐）

Windows 11 自带 winget 包管理器：

```bash
# 安装
winget install Microsoft.VisualStudioCode

# 或指定版本
winget install Microsoft.VisualStudioCode --version 1.92.0
```

> 💡 winget 自动处理 PATH 配置，无需手动设置。

### 方式三：Chocolatey 安装

使用 Chocolatey 包管理器：

```bash
# 安装 Chocolatey（如未安装）
Set-ExecutionPolicy Bypass -Scope Process -Force
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# 安装 VS Code
choco install vscode -y
```

### 方式四：Scoop 安装

```bash
# 安装 Scoop（如未安装）
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex

# 安装 VS Code
scoop bucket add extras
scoop install vscode
```

## 验证安装

```bash
# 打开命令提示符或 PowerShell，运行：
code --version

# 应该输出类似：
# 1.92.0
# x64

# 用 VS Code 打开当前目录
code .
```

## 首次启动设置

### 设置中文界面

1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展面板
3. 搜索 **Chinese (Simplified) Language Pack**
4. 点击 **Install**，安装后按提示 **Restart**

### 基础设置

打开设置：`Ctrl+,`（逗号）

推荐的首次配置：

```jsonc
// settings.json
{
  "editor.fontSize": 14,
  "editor.fontFamily": "'Cascadia Code', 'Fira Code', 'JetBrains Mono', Consolas, monospace",
  "editor.fontLigatures": true,
  "editor.minimap.enabled": true,
  "editor.tabSize": 2,
  "workbench.colorTheme": "Default Dark+",
  "files.autoSave": "afterDelay",
  "editor.formatOnSave": true,
  "terminal.integrated.defaultProfile.windows": "PowerShell"
}
```

> 💡 在 `Ctrl+Shift+P` 中搜索 `settings.json` 即可直接编辑。

### 配色主题推荐

| 主题名称 | 类型 | 特点 |
|---------|------|------|
| **Default Dark+** | 深色 | VS Code 默认，护眼 |
| **One Dark Pro** | 深色 | Atom 经典配色 |
| **GitHub Dark** | 深色 | GitHub 风格 |
| **Catppuccin** | 多种 | 柔和配色，人气最高 |

## 配置右键菜单

如果安装时没有勾选右键菜单，可以手动添加：

1. 打开 VS Code
2. `Ctrl+Shift+P` → `Shell Command: Install 'code' command in PATH`
3. 注册到文件资源管理器上下文菜单

## PowerShell 终端优化

VS Code 在 Windows 上默认使用 PowerShell。推荐优化：

```jsonc
// settings.json
{
  "terminal.integrated.defaultProfile.windows": "PowerShell",
  "terminal.integrated.profiles.windows": {
    "PowerShell": {
      "source": "PowerShell",
      "icon": "terminal-powershell",
      "args": ["-NoLogo"]
    },
    "Command Prompt": {
      "path": ["${env:windir}\\System32\\cmd.exe"],
      "icon": "terminal-cmd"
    },
    "Git Bash": {
      "path": "C:\\Program Files\\Git\\bin\\bash.exe",
      "icon": "terminal-bash"
    }
  }
}
```

## 卸载 VS Code

```bash
# 方式一：通过设置应用
# 设置 → 应用 → 应用和功能 → 搜索 "Visual Studio Code" → 卸载

# 方式二：winget
winget uninstall Microsoft.VisualStudioCode

# 方式三：Chocolatey
choco uninstall vscode -y
```

用户设置文件位置：`%APPDATA%\Code\User\`
扩展文件位置：`%USERPROFILE%\.vscode\extensions`
