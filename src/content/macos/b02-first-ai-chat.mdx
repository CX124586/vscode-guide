> 🎯 **本教程适合：** 已经装好 VS Code 和 AI 助手的同学。还不确定怎么装的？先看[搭建你的AI助手](/docs/macos/b01-beginner-setup)。

## 打开 AI 对话面板

在 VS Code 中，按下键盘上的 **`Cmd+Shift+I`**（三个键同时按）：

- `Cmd` 是空格键左边的 ⌘ 键
- `Shift` 是键盘上方的 ⇧ 键
- `I` 是字母 I

> 💡 按对后，屏幕右侧会出现一个聊天面板，这就是 AI 对话窗口。

## 和 AI 打第一声招呼

在聊天面板底部的输入框中，输入下面这段话：

```
你好，请帮我写一份简短的工作周报。
```

然后按 **回车键**（Enter）发送。

> ✅ 几秒钟后，AI 就会回复你一份周报模板。如果觉得写得不错，可以继续往下走。

## 让 AI 帮你做 3 件最常用的事

### 1. 写东西

**场景：** 你需要写一份会议邀请邮件。

在 AI 对话框中输入：

```
帮我写一封会议邀请邮件，主题是讨论第一季度项目进度。
参会人员：张总、李经理、王主管。
时间：下周三下午2点。
语气要正式但友好。
```

**结果：** AI 会立刻生成一封完整的邮件，你可以直接复制到邮件客户端发送。

### 2. 改东西

**场景：** 你有一大段文字需要润色。

先把你的原文复制下来，然后在 AI 对话框中输入：

```
请帮我润色下面这段文字，让它更通顺、更专业：

[把你的原文粘贴在这里]
```

> 💡 **如何粘贴？** 选中文字按 `Cmd+C` 复制，在 AI 输入框中按 `Cmd+V` 粘贴。

### 3. 问问题

**场景：** 你需要快速了解一个陌生概念。

直接在 AI 对话框中问：

```
请用通俗易懂的语言解释什么是"区块链"，越简单越好，不要说专业术语。
```

> ✅ AI 会用最通俗的方式回答你。你可以把 "区块链" 换成任何你想了解的概念。

## 三个实用小技巧

### 技巧一：用 `/` 调出 AI 技能

在 AI 输入框中输入 `/`，你会看到一系列 AI 技能：

| 输入 | 功能 | 适合场景 |
|------|------|---------|
| `/explain` | 解释一段文字 | 看不懂某个文件或概念时 |
| `/brainstorming` | 头脑风暴 | 需要想点子、列方案时 |
| `/help` | 查看帮助 | 忘了怎么用的时候 |

**试试看：** 输入 `/brainstorming 帮我列出10个周末团建活动的点子`

### 技巧二：指定 AI 的角色

让 AI 扮演特定角色，回答会更精准：

```
你是一名有10年经验的HR专家。
我现在要写一份招聘文案，招聘一名新媒体运营。
请帮我写出招聘要求的核心内容。
```

### 技巧三：告诉 AI 输出格式

你希望 AI 的输出是什么格式，直接告诉它：

```
请用表格形式列出：北京、上海、广州、深圳 这四个城市的特点对比。
包括：人口、气候、房价水平、饮食特色 四个维度。
```

## 进阶：使用 Superpowers 工作流

如果你安装了 Superpowers（参考 [搭建指南](/docs/macos/b01-beginner-setup#第五步安装-superpowersai-对话增强)），你还可以用更强大的命令：

### 案例：让 AI 帮你制定计划

在 AI 中输入：

```
/plan 我下周要做一个客户汇报，需要准备PPT大纲、演讲逐字稿、以及可能的问题应答。请帮我出一份详细的准备计划。
```

### 案例：让 AI 帮你头脑风暴

```
/brainstorming 公司年会需要设计一个互动环节，预算5000元以内，参与人数50人左右。有什么创意方案？
```

## 常见问题

**Q: AI 的回答不准确怎么办？**
A: 重新描述你的需求，用更具体的说法。比如不说"帮我写个通知"，而是说"帮我写一份关于周五下午3点全部门会议的通知"。

**Q: AI 能看懂中文吗？**
A: 当然可以！用中文提问，AI 会用中文回答。

**Q: 写的内容可以保存吗？**
A: 可以。AI 的回答可以像普通文字一样选中、复制（`Cmd+C`），然后粘贴到任何地方（Word、邮件、微信等）。
