掌握快捷键是提升 VS Code 使用效率最直接的方式。以下按功能分类整理。

## 最核心快捷键（先记住这 10 个）

| 快捷键 | 功能 |
|--------|------|
| `Cmd+Shift+P` | 命令面板（记住这一个就能做所有事） |
| `Cmd+P` | 快速打开文件 |
| `Cmd+B` | 切换侧边栏 |
| `Cmd+J` | 切换面板（终端/输出等） |
| `Cmd+\` | 分屏 |
| `Cmd+W` | 关闭当前标签页 |
| `Cmd+Shift+K` | 删除当前行 |
| `Cmd+D` | 选中当前词（继续按选中下一个相同词） |
| `Option+↑/↓` | 向上/下移动当前行 |
| `Cmd+Shift+Enter` | 在当前行上方插入空行 |

> 💡 **习惯养成**：不必一次全记住，每次刻意练习一个新快捷键，一周就能形成肌肉记忆。

## 自定义快捷键

VS Code 允许你自由修改快捷键绑：

1. 按 `Cmd+Shift+P` 搜索 `keyboard shortcuts`
2. 打开 `keybindings.json`
3. 添加自定义绑定：

```jsonc
// keybindings.json
[
  {
    "key": "cmd+shift+n",
    "command": "explorer.newFolder"
  },
  {
    "key": "alt+cmd+t",
    "command": "workbench.action.terminal.toggleTerminal"
  }
]
```

> 💡 建议只自定义最常用的 3-5 个操作，其他的先适应默认快捷键。
