VS Code 的强大很大程度上来自其丰富的扩展生态。以下按类别推荐经过验证的扩展：

## 扩展管理

| 操作 | 快捷键 |
|------|--------|
| 打开扩展面板 | `Cmd+Shift+X` |
| 搜索扩展 | 在扩展面板搜索框输入 |
| 安装扩展 | 点击 Install |
| 卸载扩展 | 右键 → Uninstall |
| 禁用扩展 | 右键 → Disable |
| 从命令行安装 | `code --install-extension <id>` |

### 批量安装

你可以通过命令行批量安装常用扩展：

```bash
# 一键安装推荐扩展
code --install-extension esbenp.prettier-vscode
code --install-extension dbaeumer.vscode-eslint
code --install-extension bradlc.vscode-tailwindcss
# ...
```

## 扩展配置示例

### Prettier

```jsonc
// .prettierrc
{
  "semi": false,
  "singleQuote": true,
  "trailingComma": "all",
  "printWidth": 100,
  "tabWidth": 2,
  "arrowParens": "always"
}
```

### ESLint

```jsonc
// settings.json
{
  "eslint.validate": ["javascript", "typescript", "javascriptreact", "typescriptreact"],
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": "explicit"
  }
}
```

## 扩展性能管理

装太多扩展会影响 VS Code 的启动速度和运行流畅度：

```jsonc
// settings.json
{
  // 只在特定语言下启用某些扩展
  "python.disableInstallationCheck": true,

  // 禁用不常用的扩展
  "extensions.ignoreRecommendations": false,

  // 查看启动时间：Cmd+Shift+P → Developer: Show Running Extensions
}
```

**查看扩展对性能的影响：**

运行 `> Developer: Show Running Extensions`，可以看到每个扩展的启动耗时。

## 扩展推荐导出与分享

```bash
# 导出已安装列表
code --list-extensions > vscode-extensions.txt

# 从文件批量安装
cat vscode-extensions.txt | xargs -L 1 code --install-extension
```
