## 系统要求

VS Code 支持 macOS **10.15 (Catalina)** 及以上版本，包括 Apple Silicon (M1/M2/M3/M4) 和 Intel 芯片的 Mac。

## 安装方式

### 方式一：官网下载（推荐）

1. 访问 [VS Code 官网](https://code.visualstudio.com)
2. 点击 **Download for macOS** 按钮
3. 将下载的 `Visual Studio Code.zip` 解压
4. 将 `Visual Studio Code.app` 拖入 **Applications** 文件夹
5. 首次打开时，macOS 可能提示"无法验证开发者"，点击 **打开** 即可

### 方式二：Homebrew 安装

如果你使用 Homebrew，这是最便捷的方式：

```bash
# 安装
brew install --cask visual-studio-code

# 验证安装
code --version
```

### 方式三：命令行安装

VS Code 也可以完全通过命令行管理：

```bash
# 使用 brew 安装
brew install --cask visual-studio-code

# 安装后，在 VS Code 中打开命令面板（Cmd+Shift+P）
# 搜索 "Shell Command: Install 'code' command in PATH" 并执行
# 完成后即可在终端中使用 code 命令
```

验证命令行是否配置成功：

```bash
code --version
# 应输出类似：1.92.0
code .
# 在当前目录启动 VS Code
```

## 首次启动设置

### 设置中文界面

1. 打开 VS Code
2. 点击左侧活动栏的 **Extensions** 图标（或按 `Cmd+Shift+X`）
3. 搜索 **Chinese (Simplified) Language Pack**
4. 点击 **Install**
5. 安装完成后按提示 **Restart**

### 基础设置

打开设置界面：`Cmd+,`（逗号）

建议首次调整的配置项：

```jsonc
// settings.json
{
  "editor.fontSize": 15,
  "editor.fontFamily": "'Cascadia Code', 'Fira Code', 'JetBrains Mono', monospace",
  "editor.fontLigatures": true,
  "editor.minimap.enabled": true,
  "editor.tabSize": 2,
  "workbench.colorTheme": "Default Dark+",
  "files.autoSave": "afterDelay",
  "editor.formatOnSave": true,
  "editor.renderWhitespace": "boundary"
}
```

> 💡 **提示**：在 `Cmd+Shift+P` 中搜索 `settings.json` 即可打开配置文件直接编辑。

### 配色主题推荐

| 主题名称 | 类型 | 特点 |
|---------|------|------|
| **Default Dark+** | 深色 | VS Code 默认，护眼 |
| **Default Light+** | 浅色 | 明亮环境适用 |
| **One Dark Pro** | 深色 | Atom 经典配色 |
| **GitHub Dark** | 深色 | GitHub 风格 |
| **Catppuccin** | 多种 | 柔和配色，人气最高 |

## 验证安装

创建一个测试文件来确认一切正常：

```bash
# 创建测试目录
mkdir -p ~/vscode-test
cd ~/vscode-test

# 用 VS Code 打开
code .

# 新建文件 test.py 并输入：
# print("Hello VS Code!")
```

如果 VS Code 正常打开并显示文件内容，安装成功！

## 卸载 VS Code

如果需要卸载：

```bash
# 方法一：从 Applications 中删除
rm -rf /Applications/Visual Studio Code.app

# 方法二：使用 Homebrew
brew uninstall --cask visual-studio-code
```

用户设置文件位置：`~/Library/Application Support/Code/User/`
扩展文件位置：`~/.vscode/extensions/`
