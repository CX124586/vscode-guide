## VS Code 界面组成

VS Code 的界面主要分为以下五大区域：

```
┌─────────────┬──────────────────────────────────────┬──────────┐
│   活动栏     │              编辑器组                   │  侧边栏   │
│  (Activity  │  (Editor Groups)                      │ (Side    │
│   Bar)     │                                       │  Bar)    │
│             │                                       │          │
│  ☰ 文件     │   ［标签页1］ ［标签页2］                │  ├ 目录   │
│  🔍 搜索    │                                       │  ├ 大纲   │
│  ⎔ 源码管理  │   ┌────────────────────────────────┐  │  └ ...   │
│  🐞 调试    │   │  你的代码编辑区域                  │  │          │
│  ■ 扩展     │   │                                  │  │          │
│             │   └────────────────────────────────┘  │          │
│             │                                       │          │
├─────────────┴──────────────────────────────────────┴──────────┤
│                        面板 (Panel)                            │
│        终端 │ 调试控制台 │ 问题 │ 输出 │ 端口                    │
└──────────────────────────────────────────────────────────────┘
│                        状态栏 (Status Bar)                      │
│  分支: main  │  1 ⨯ 0 △ 0 ◼  │  UTF-8  │  Go  │  行 10, 列 5  │
└──────────────────────────────────────────────────────────────┘
```

### 1. 活动栏 (Activity Bar)

位于最左侧，是 VS Code 的主要导航入口：

| 图标 | 名称 | 快捷键 | 说明 |
|------|------|--------|------|
| ☰ | 文件资源管理器 | `Cmd+Shift+E` | 浏览和管理项目文件 |
| 🔍 | 搜索 | `Cmd+Shift+F` | 全文搜索和替换 |
| ⎔ | 源码管理 | `Cmd+Shift+G` | Git 集成 |
| 🐞 | 运行与调试 | `Cmd+Shift+D` | 调试配置 |
| ■ | 扩展 | `Cmd+Shift+X` | 扩展市场 |
| 👤 | 账户 | — | 登录同步设置 |

> **技巧**：你可以右键活动栏自定义显示哪些图标，也可以拖拽调整顺序。

### 2. 侧边栏 (Side Bar)

显示当前活动栏选中的内容。最常见的视图：

- **文件资源管理器**：项目文件树，支持 `展开/折叠` `拖拽移动` `右键菜单`
- **搜索**：支持正则搜索、文件包含/排除、替换
- **Git**：变更列表、暂存、提交、分支切换

**快捷键操作：**
- `Cmd+B` — 切换侧边栏显示/隐藏
- `Cmd+Shift+E` — 聚焦文件资源管理器
- `Cmd+Shift+F` — 聚焦搜索

### 3. 编辑器组 (Editor Groups)

代码编辑的核心区域，支持**分屏**和**标签页**管理：

**分屏操作：**
- `Cmd+\` — 垂直分屏
- `Cmd+Shift+\` — 水平分屏（需安装扩展）
- `Cmd+1/2/3` — 切换到第 N 个编辑器组

**标签页操作：**
- `Cmd+W` — 关闭当前标签
- `Cmd+Shift+T` — 恢复关闭的标签
- `Cmd+Option+→/←` — 切换标签页
- `Cmd+Option+数字` — 跳转到第 N 个标签

### 4. 面板 (Panel)

底部区域，可容纳多个工具：

| 面板 | 快捷键 | 用途 |
|------|--------|------|
| 终端 | `` Ctrl+` `` | 集成终端 |
| 调试控制台 | `Cmd+Shift+Y` | 调试输出 |
| 问题 | `Cmd+Shift+M` | 错误和警告列表 |
| 输出 | — | 扩展日志 |

### 5. 状态栏 (Status Bar)

底部状态信息，随当前上下文变化：

从左到右依次显示：
- **分支信息** — 当前 Git 分支
- **错误/警告计数** — ⨯ 错误 △ 警告 ◼ 信息
- **编码格式** — UTF-8、GBK 等
- **光标位置** — 行号：列号
- **缩进设置** — 空格/制表符
- **语言模式** — 当前文件语言

> **技巧**：点击状态栏的任意项都可以快速切换设置！

## 命令面板 (Command Palette)

命令面板是 VS Code 最强大的入口，**所有操作都可以通过命令面板完成**。

- **打开：** `Cmd+Shift+P`
- 输入命令名称即可执行

```bash
# 常用的命令面板操作
> settings.json          # 打开设置文件
> install extension      # 安装扩展
> theme                 # 切换配色主题
> keybinding           # 打开快捷键设置
> terminal: create     # 创建新终端
> git: commit         # Git 提交
```

> 💡 **习惯使用命令面板**是成为 VS Code 高手的第一个习惯。记不住的快捷键都可以用命令面板代替。

## 文件操作基础

### 打开文件夹

- `Cmd+O` — 打开文件
- `code .` — 终端中打开当前目录
- 拖拽文件夹到 VS Code 图标 — 直接在 sidebar 中打开

### 文件快捷操作

| 操作 | 快捷键 |
|------|--------|
| 快速打开文件 | `Cmd+P` |
| 在两文件间切换 | `Ctrl+Tab` |
| 保存 | `Cmd+S` |
| 全部保存 | `Cmd+Option+S` |
| 关闭当前 | `Cmd+W` |
| 关闭全部 | `Cmd+K W` |

## 工作区 (Workspace)

工作区是 VS Code 管理项目的高级方式：

```json
// example.code-workspace
{
  "folders": [
    { "name": "前端", "path": "../frontend" },
    { "name": "后端", "path": "../backend" }
  ],
  "settings": {
    "editor.fontSize": 14,
    "editor.tabSize": 2
  }
}
```

- **单文件夹工作区**：直接用 `code .` 打开
- **多根工作区**：通过 `.code-workspace` 文件管理多个项目
- 工作区可以包含**独立配置**，不影响全局设置
