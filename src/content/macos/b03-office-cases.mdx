> 🎯 **怎么用：** 找到你需要的场景，复制灰色框里的文字，粘贴到 AI 对话框中（`Cmd+Shift+I`），**按回车**，AI 就会帮你搞定。
>
> 英文是原本的命令格式，建议先输入指令再在后面自由补充。所有提示词都附有具体示例，方便替换为你自己的内容。

## 📝 写作类

### 写周报 / 日报

选中下面的文字，按 `Cmd+C` 复制，在 AI 对话框中按 `Cmd+V` 粘贴，然后把括号内的内容换成你的：

```
请帮我写一份本周工作总结。
本周完成了以下工作：
- （这里写你做了什么，比如：整理了客户档案）
- （比如：参加了部门周会）
- （比如：完成了季度报表）

请按以下格式输出：
【本周完成】
【存在问题】  
【下周计划】
```

> 💡 **效果：** AI 会把散乱的要点整理成结构清晰的工作总结，直接复制到周报里即可。

### 写邮件

```
请帮我写一封商务邮件。
收件人：王经理
主题：关于项目延期的沟通
要点：
1. 由于供应商交货延迟，项目预计延期2周
2. 我们已经找好了备选方案
3. 希望周三前开个电话会沟通
语气：诚恳、专业、不推卸责任
```

### 写会议通知

```
请帮我写一份部门会议通知。
会议主题：Q2 营销方案讨论
时间：本周四下午 3:00-4:00
地点：3楼大会议室
参会人员：市场部全员
会议议程：
1. 上季度数据回顾（15分钟）
2. 新方案提案（30分钟）
3. 自由讨论（15分钟）
请写成正式的邮件格式。
```

### 写请假申请

```
请帮我写一份请假申请。
姓名：张三
请假时间：2024年6月10日-6月12日（共3天）
请假原因：个人年假，回老家办事
工作交接：已委托李四代为处理紧急事务
请写成正式的书面格式。
```

## 🔄 润色与改写

### 润色一段文字

把你想润色的原文，**替换掉**下面括号里的内容：

```
请帮我润色下面这段文字，让它读起来更流畅、更专业：

（把你的原文粘贴在这里）
```

### 把长文缩短

```
请把下面这段文字压缩到200字以内，保留核心信息：

（把长文粘贴在这里）
```

### 把一段话改成 PPT 要点

```
请把下面这段文字改写成PPT bullet points格式，每点不超过15个字：

（把文字粘贴在这里）
```

## 📋 整理与分析

### 整理会议纪要

```
请帮我整理一份会议纪要。

会议主题：产品需求评审
时间：2024年6月5日
参会人：产品部、研发部、设计部

讨论内容：
（这里写下你的会议笔记，即使是零散的也行）

请整理成：【会议目的】【讨论要点】【决议事项】【待办事项】四个部分。
```

> 💡 **小技巧：** 会议中先用记事本随便记下关键词，会后扔给 AI 整理，省时省力。

### 提取关键信息

```
请从下面这段文字中提取关键信息，用表格形式输出：
- 涉及的人物
- 时间节点
- 金额
- 行动项

（粘贴文字到这里）
```

### 分析优劣势

```
请帮我分析以下方案的优劣势。

方案名称：（填写方案名称）
方案内容：（描述方案内容）

请用 SWOT 分析框架（优势、劣势、机会、威胁）来输出。
```

## 📊 数据处理

### 把文字转成表格

```
请把下面这段文字整理成表格：

一季度营收500万，同比增长15%，其中华东区贡献最大占比40%，华北区30%。二季度目标600万，华东区目标250万，华北区目标200万。

表格格式：| 项目 | 数值 | 备注 |
```

### 提取并归类

```
请把下面这段内容中的项目按类别归类：

（粘贴杂乱的内容到这里）

归类标准：按「已完成」「进行中」「待启动」三类整理成表格。
```

## 💡 创意类

### 头脑风暴

```
请帮我做一轮头脑风暴，主题是：
（写下你的主题，比如：公司团建活动的创意方案）

请输出10个不同的想法，每个想法用一句话描述。
```

### 写活动方案大纲

```
请帮我写一份公司团建活动方案大纲。
参与人数：30人
预算：每人500元
时长：一天
要求：包含户外活动+室内互动
```

### 起名字 / 起标题

```
请帮我想10个微信公众号文章标题，主题是：（填写主题）
要求：吸引人点击，但不夸张，适合职场人群。
```

### 快速学习新概念

```
请用费曼学习法（用最简单的语言）解释什么是：

（填写你想了解的概念，比如：什么是OKR、什么是私域流量等）

要求：不使用专业术语，用生活中的例子打比方。
```

## 📱 这些案例可以组合使用

很多场景可以串联起来：

**案例：从会议到周报**

```
第一步：/brainstorming 下周部门会议的议题建议

第二步：把第一步的结果完善后生成会议通知

第三步：会议结束后，把笔记扔给AI整理纪要

第四步：把纪要和下阶段计划塞进周报模板
```

**案例：从资料到汇报PPT**

```
第一步：把一堆杂乱资料扔给AI，让它整理出要点

第二步：让AI把要点写成PPT大纲

第三步：让AI把每页大纲扩展成逐字稿

第四步：让AI翻译成英文版（如果需要）
```

## 让你的案例被记住——保存技巧

AI 对话内容不会自动保存。如果你觉得某次回答特别好：

1. 用鼠标选中 AI 的回答
2. 按 **`Cmd+C`** 复制
3. 打开一个空白的 Word 文档或备忘录
4. 按 **`Cmd+V`** 粘贴保存

> 💡 **进阶技巧：** 你也可以在 VS Code 中按 **`Cmd+N`** 新建一个文件，粘贴进去，然后按 **`Cmd+S`** 保存到电脑上。
