> 🎯 **本教程适合：** 完全零基础的小白。不需要懂任何编程知识，只要能操作鼠标和键盘就行。

## 第一步：下载 VS Code

VS Code 是一个**免费**的文字编辑软件，我们将用它来和 AI 对话。

**跟着做：**

1. 打开你的 Mac 上的浏览器（Safari 或 Chrome 都可以）
2. 在地址栏输入网址：`https://code.visualstudio.com`
3. 进入网站后，你会看到一个蓝色的 **Download for macOS** 按钮
4. 点击这个按钮，浏览器会自动下载一个 ZIP 压缩文件

> 🔍 **找不到按钮？** 网页打开后，正中间最大的那个蓝色按钮就是。

## 第二步：安装 VS Code

1. 下载完成后，打开你的 **下载** 文件夹（通常在 Dock 栏最右边有个向下箭头图标）
2. 找到 `Visual Studio Code.zip` 文件，双击它
3. 解压后会看到一个名为 `Visual Studio Code` 的应用程序（图标是蓝白色的）
4. **用鼠标把它拖进「应用程序」文件夹**（左边栏的 Applications）
5. 打开「应用程序」文件夹，找到 `Visual Studio Code`，双击打开

> ⚠️ **第一次打开时**，Mac 可能会提示"无法验证开发者"。别担心，点击 **「打开」** 按钮就可以了。

## 第三步：安装中文语言包

VS Code 默认是英文界面，我们把它变成中文。

**跟着做：**

1. 打开 VS Code 后，你会看到主界面
2. 点击左侧竖排图标中的 **最下面那个**（四个方块组成的小图标），鼠标悬停会显示 "Extensions"
3. 在顶部搜索框中，输入：`Chinese`
4. 搜索结果中第一个应该是 **Chinese (Simplified) Language Pack**
5. 点击右边的 **Install**（安装）按钮
6. 等待几秒钟，安装完成后点击右下角弹出的 **Restart**（重启）
7. VS Code 会重新打开，界面已经变成中文了！🎉

## 第四步：安装 GitHub Copilot（AI 对话功能）

这是最关键的一步——安装 AI 助手。

**跟着做：**

1. 仍然在 VS Code 中，点击左侧的「扩展」图标（四个方块）
2. 在搜索框中输入：`GitHub Copilot`
3. 你会看到 **GitHub Copilot** 这个扩展，点击它
4. 点击 **Install** 按钮
5. 安装完成后，VS Code 右下角会弹出提示让你登录 GitHub 账号

> 💡 **不想用 GitHub Copilot？** 也可以选择国产 DeepSeek 大模型，更便宜、国内网络直接访问，详见[配置 DeepSeek：用国产大模型当 AI 大脑](/docs/macos/b04-model-config)。

> 💡 **没有 GitHub 账号？** 接下来教你注册一个——完全免费。

### 注册 GitHub 账号

1. 打开浏览器，访问 `https://github.com`
2. 在首页的注册表单中填写：
   - **Username**（用户名）：给自己取一个英文名字，比如 `zhangsan2024`
   - **Email**（邮箱）：输入你的常用邮箱
   - **Password**（密码）：设置一个密码（至少 8 位，含字母和数字）
3. 点击 **Create account**
4. 去你的邮箱查收验证邮件，点击其中的验证链接
5. 按提示完成验证（选择 Free 免费计划即可）

### 在 VS Code 中登录

1. 回到 VS Code，点击右下角的 **Sign in to GitHub** 按钮
2. 浏览器会自动打开，点击 **Authorize Visual Studio Code** 授权
3. 授权完成后，回到 VS Code 即可

## 第五步：安装 Superpowers（AI 对话增强）

Superpowers 让 AI 变得更聪明，能帮你写周报、翻译、做总结等。

**跟着做：**

1. 在 VS Code 顶部菜单栏点击 **查看（View）** → **命令面板（Command Palette）**
2. 或者直接按快捷键：**`Cmd+Shift+P`**
3. 在弹出的框中输入：`> install extensions from VSIX`
4. 但更简单的方式是按 **`Cmd+Shift+I`** 打开 AI 对话面板
5. 在对话输入框左边，点击设置图标 ⚙️ → **Plugins（插件）**
6. 点击 **从源安装插件**
7. 输入以下地址后按回车：
   ```
   https://github.com/obra/superpowers
   ```
8. 等待安装完成

> ✅ **验证是否成功：** 在 AI 对话框中输入 `/help`，如果看到一大串命令列表，说明安装成功了！

## 第六步：用你的 AI 助手做第一件事

现在一切都准备好了！让我们试试 AI 的能力：

1. 按 **`Cmd+Shift+I`** 打开 AI 对话面板
2. 在输入框中输入：
   ```
   请帮我写一段话：今天是2024年1月15日，我需要向领导汇报本周工作进展，完成了3个项目文档的整理，参加了两场会议，下周计划开始新项目调研。
   ```
3. 按回车发送

> 🎉 **恭喜！你已经成功用上了 AI 助手！** 接下来去看看「职场案例库」，里面有更多可以直接复制粘贴使用的例子。

## 遇到问题怎么办？

| 问题 | 解决方法 |
|------|---------|
| 下载速度慢 | 换一个网络环境，或尝试用 Chrome 浏览器下载 |
| 安装时提示"无法打开" | 去 **系统设置 → 隐私与安全性**，点"仍要打开" |
| 登录 GitHub 卡住 | 检查你的网络能否访问 github.com，可能需要开 VPN |
| AI 不回消息 | 检查右下角 GitHub Copilot 图标是否亮起，点击可以重新登录 |
